# Design Lead is on vacation — Pixso Plugin

Чек-лист дизайнера с автоматической проверкой контраста по **WCAG 2.1** (SC 1.4.3).

## Что делает плагин

### Чек-лист (40 вопросов)

Интерактивный опрос в формате **Да / Нет** по критериям UX, доступности и передачи в разработку.

- Ответ **Нет** добавляет пункт в список **Доработки** с описанием, что исправить
- К каждой доработке можно добавить комментарий
- Прогресс сохраняется в `localStorage` плагина
- Список доработок можно скопировать в буфер обмена

### Проверка контраста

Встроена в чек-лист и доступна в любой момент:

- Ссылка **Проверить контраст** в шапке вопроса — быстрая проверка без завершения чек-листа
- Последний вопрос чек-листа — финальная проверка всех макетов на странице
- На финальном экране: **Доработки** + **Контраст** (проблемы / пройдено)
- Клик по карточке контраста выделяет текстовый слой на канвасе

Пороги WCAG AA:

| Тип текста | Минимум |
|---|---|
| Обычный | **4.5:1** |
| Крупный (≥24px или ≥18.5px жирный) | **3:1** |

Также показываются пороги **AAA** (7:1 / 4.5:1) и короткие рекомендации по исправлению.

## Setup

```bash
npm install
npm run build
npm test
```

## Development

```bash
npm run watch
```

Пересобирает `code.js`, `ui.js` и `ui.html` при изменениях.

## Install in Pixso

1. **Plugins → Development → Import plugin**
2. Выберите `manifest.json` в этой папке
3. **Plugins → Design Lead is on vacation → Open Design Lead is on vacation**

> **Примечание:** `id` плагина — `com.pixso.design-lead-is-on-vacation`. После смены id нужно переимпортировать плагин; прогресс чек-листа хранится под ключом `design-lead-checklist-v2`.

## Ограничения

**Контраст**

- Сканируются все top-level макеты на текущей странице
- Если на странице нет макетов или текстовых слоёв — показывается понятное сообщение
- Фон определяется по сплошным заливкам соседних слоёв, предков и фона страницы
- Градиенты, изображения и эффекты на фоне дают приблизительный результат с предупреждением
- Градиентная или растровая заливка текста не анализируется

**Чек-лист**

- Ручная самопроверка; ответы не синхронизируются с файлом Pixso
- Отчёт контраста сохраняется в `sessionStorage` до закрытия вкладки плагина
